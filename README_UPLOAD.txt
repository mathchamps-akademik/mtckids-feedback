# Raw GitHub - Form Input MTC Kids

Upload seluruh isi folder ini ke repository GitHub Pages khusus form input.

File utama:
- index.html

Contoh URL setelah publish:
- https://mathchamps-akademik.github.io/NAMA-REPO-FORM/

Fungsi:
- MTC Kids mengisi Nama MTC Kids, Cabang homebase, Program yang berkaitan, saran/kendala, contoh kejadian, dukungan yang dibutuhkan, dan prioritas.
- Pilihan Program yang berkaitan: SG Math, Sempoa, SG Math & Sempoa.
- Hasil submit tersimpan online ke storage yang sama dengan halaman Data Hasil Form.
